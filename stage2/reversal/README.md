# Block-Reversal Task

**Статус:** 🟡 В разработке

## 📖 Описание

Задача на когнитивную гибкость и персеверацию. Агент обучается правилам (Left = 80%, Right = 20%), затем правила скрыто инвертируются (Left = 20%, Right = 80%). Измеряется скорость адаптации и количество персеверативных ошибок.

### Структура блоков

```
Блок 1 (Триалы 1-100):   Left = 80%, Right = 20%
          │
          ▼
    [REVERSAL]
          │
          ▼
Блок 2 (Триалы 101-200): Left = 20%, Right = 80%
```

## 📊 Ожидаемая динамика

| Фаза | Поведение агента | V_G | V_p |
| :--- | :--- | :--- | :--- |
| **Блок 1 (стабильность)** | EXPLOIT, высокая точность | Растёт → 0.8 | Растёт → 0.8 |
| **Сразу после реверсала** | Персеверация, ошибки | Высокий (инерция) | Высокий (инерция) |
| **Расплавление** | EXPLORE, тестирование | Падает → 0.2 | Падает → 0.2 |
| **Блок 2 (адаптация)** | EXPLOIT, точность растёт | Растёт → 0.8 | Растёт → 0.8 |

## 📈 Метрики

| Метрика | Описание | Ожидаемый паттерн |
| :--- | :--- | :--- |
| **Trials to Criterion (TTC)** | Триалы до 80% точности после реверсала | Full > NoVG |
| **Perseverative Errors** | Повторения старого выбора после реверсала | Full > NoVp > NoVG |
| **Exploration Rate** | % EXPLORE в фазе 101-130 | NoVG > Full > NoVp |
